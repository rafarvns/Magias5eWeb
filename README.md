# Grimório D&D 5E

Aplicação web para visualização e consulta de magias da 5ª Edição de Dungeons & Dragons.

## Funcionalidades

- **Visualização em Grade**: Interface moderna para busca e consulta rápida de magias.
- **Modo de Impressão (A4)**: Formatação otimizada para impressão de listas de magias em folhas A4.
- **Cartas de Magia (A7)**: Geração de cartas de magia no formato A7 para uso em mesa.
- **Filtros Avançados**: Filtre magias por classe e nível.
- **Busca Instantânea**: Pesquise magias pelo nome.

## Como usar

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório.
3. Instale as dependências (se necessário):
   ```bash
   npm install
   ```
4. Inicie o servidor:
   ```bash
   npm start
   ```
5. Acesse `http://localhost:3000` no seu navegador.

## Créditos e Fontes

Este projeto foi desenvolvido de forma independente, utilizando recursos da comunidade de D&D 5E:

- **Tradução**: Conteúdo traduzido por AlanVenic Gonçalves e Grupo Material RPG, disponibilizado pela [Biblioteca Élfica](http://www.bibliotecaelfica.com/).
- **Interface e Estrutura**: Inspirado na organização e dados do projeto [magias5e](https://avribacki.gitlab.io/magias5e/).
- **Ícones**: Criados por diversos autores (Alfa Design, Creative Stall, Nikita Kozin, entre outros) via [The Noun Project](https://thenounproject.com).
